diving in

exercise instructions (groups of 2-4)

1. Download the exercise code.

2. Setup a local environment. Only static hosting is relevant so you can use any framework you like (the project comes with a node express server).

3. Play around, draw some shapes.

4. Play break-golf twice: make together some change to the code, then compete to find the change's effect(s).

5. Find the bug (more accurately, the inconsistent behaviour). Fix it.

6. Add a button for drawing triangles.

7. When picking points (for all shapes), make markers to signal previously picked points. These markers should disappear when the shape is drawn.

8. Add a button allowing the user to pick a color for the shapes.

9. Enjoy some well-earned doodling.

10. Understand how handle_drawing works.
